# 🌦 React Weather Dashboard

A modern weather dashboard built using React JS and OpenWeather API.

## 🚀 Features

- Live weather data
- City search suggestions
- Save favorite city
- Auto-load saved city weather
- Responsive UI
- Dynamic weather icons
- Local storage support
- Modern glassmorphism design

## 🛠 Technologies Used

- React JS
- CSS3
- OpenWeather API
- JavaScript
- Vite
